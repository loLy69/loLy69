## Featured projects

### Manicure Booking Bot

Commercial Telegram bot template for appointment-based businesses. Clients can select a service, choose an available time and manage their bookings. Administrators can manage services, appointments and notifications.

**Stack:** Python, aiogram 3, SQLAlchemy, PostgreSQL, Alembic, Docker

[View repository →](https://github.com/loLy69/manicure-booking-bot-template)

---

### CDO SGUPS Telegram Bot

Telegram bot for an educational centre with separate roles for students, parents, teachers and administrators. The system includes registration, schedules, homework, attendance, payments, notifications and an administrative panel.

**Stack:** Python, aiogram 3, SQLite, aiohttp, Docker

[View repository →](https://github.com/loLy69/cdo-sgups-bot)

---

### MindSpark

Telegram bot template for schools, educational centres and tutors. Supports course enrolment, class schedules, homework, payments, parent–student accounts and administrative management.

**Stack:** Python, aiogram 3, SQLite, aiohttp, Docker

[View repository →](https://github.com/loLy69/st-bot)

Telegram bot for tracking planned and completed purchases using natural text input. Includes budgeting, categories, priorities, statistics and CSV export.

**Stack:** Python, aiogram 3, SQLite
